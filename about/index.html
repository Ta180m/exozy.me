About


The exozyme project is a project to build a better, privacy-respecting web and cloud-hosting service. Currently, exozyme is invite-only.


You can find all our source code at our Gitea. All HTML and CSS was hand-written with ❤️ by Ta180m.



exozyme is completely free to use! However, there are no uptime guarentees. The server is usually up between 8:30 to 22:30 CST, but it could go down at any time.
