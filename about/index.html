About


The exozyme project is a project to build a better, privacy-respecting web and cloud-hosting service. If you are interested, shoot me a message on Matrix @ta180m:exozy.me explaining what you plan to do with exozyme. We also have a Matrix space as well that you should join.


If you're interested in the technical details, the server uses a 6-core 12-thread AMD Ryzen 5 5600X proccessor paired with the 2011's finest ATI Radeon HD 6570 graphics card. We currently only have 16GB of RAM, but plenty of swap space so you don't have to worry about running out of memory. The server runs Arch Linux, but you can use a virtual machine or container to run a different OS if you would like.


Please take a look at the frequently unasked questions too.


You can find all our source code at our Gitea. All HTML and CSS was hand-written with ❤️ by Ta180m.



exozyme is completely free to use! However, there are no uptime guarentees. The server is usually up between 8:30 to 22:30 CST, but it could go down at any time.
