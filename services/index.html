Services

Core

Signing up for exozyme will automatically give you access to these services and all the desktop apps and games installed on the server.


SSH: Remote login to a terminal


xrdp: Remote desktop


Pages: Static web hosting


Web

We host a number of awesome web services that can be accessed using your Keycloak account. Ask me on Matrix if you don't have one. Note that this is a different account than what you use for the core services like remote desktop.


Keycloak: Manage your Keycloak account


Nextcloud: Cloud storage and productivity suite


Synapse: Matrix chat server, bridged to Discord


Gitea: Code hosting service


Jellyfin: Media streaming server


Mastodon: The best non-degenerate social media network


PeerTube: Watch peer-to-peer videos


MediaWiki: Wiki service used by Wikipedia


Collabora Online: Real-time collaborative editing server, integrated with Nextcloud
